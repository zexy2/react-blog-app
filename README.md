# Postify Blog

Blog app built with React 19 and Redux Toolkit. Uses JSONPlaceholder API for demo data.

**Demo:** https://zexy2.github.io/react-blog-app

## Setup

```bash
npm install
npm run dev
```

Opens at http://localhost:5173

## Features

- Post CRUD with rich text editor (TipTap)
- Bookmarks (localStorage)
- Multi-language (TR/EN)
- Analytics dashboard
- Dark/Light theme
- PWA support

## Tech

- React 19, Vite, React Router
- Redux Toolkit, TanStack Query
- TipTap editor, Recharts
- Vitest for testing

## Scripts

| Command | Description |
|---------|-------------|
| `npm run dev` | Dev server |
| `npm run build` | Production build |
| `npm run test` | Run tests |
| `npm run lint` | ESLint |
